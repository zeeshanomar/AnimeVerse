# AnimeVerse

My first HTML project while learning Web Development.

I’m currently a 1st year B.Tech CSE student, and this project is mainly me experimenting with HTML through project-based learning instead of only watching tutorials.

The goal of this project is not to make something perfect right now, but to learn by building, breaking, debugging, and improving things manually.

---

## What I Practiced Here

- HTML Boilerplate
- Headings & Paragraphs
- Links & Images
- Lists
- Tables
- Forms
- Audio & Video Tags
- Iframes
- Semantic Tags
- HTML Entities
- Multi-page Website Structure
- Basic CSS & JS File Linking
