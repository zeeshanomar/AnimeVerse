# AnimeVerse

A beginner multi-page website project built using basic HTML as part of my project-based learning journey in Web Development.

I am currently a first-year B.Tech CSE student, and this project was created to practice and strengthen my HTML fundamentals before moving deeper into CSS and JavaScript.

---

## Purpose of This Project

This project was built to:

* practice HTML through actual implementation
* improve understanding of webpage structure
* revise HTML concepts practically instead of only watching tutorials
* gain hands-on experience with building multi-page websites
* learn basic project structure and file organization
* make my first GitHub project

---
