# AnimeVerse

A beginner multi-page anime-themed website built as part of my project-based learning journey in Web Development.

I am currently a first-year B.Tech CSE student, and this project was created to practice and strengthen my HTML and CSS fundamentals before moving on to JavaScript and more advanced web technologies.

---

## Purpose of This Project

This project was built to:

* practice HTML and CSS through actual implementation
* improve understanding of webpage structure and styling
* apply responsive design concepts
* gain hands-on experience with building multi-page websites
* learn project structure and file organization
* practice Git and GitHub workflow
* build a long-term project that can be improved over time

---

## Features

* Multi-page website
* Responsive design for desktop and mobile devices
* Anime recommendation homepage
* Watch page with embedded anime trailers
* Contact form page
* Animated floating logo
* Navigation system between pages
* Audio and video integration
* Hover effects and basic animations
* Grid and Flexbox layouts
